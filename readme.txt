== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `Facebook ogg Meta Tags` plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress 

== Frequently Asked Questions == 

= Why Facebook ogg Meta Tags plugin does not appear's any where on my admin area? =

Facebook ogg Meta Tags will not add any menu on your admin area, you can check the Facebook ogg Meta Tags on your page code source [ CTR+U ]
